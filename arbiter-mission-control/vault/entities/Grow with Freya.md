# Grow with Freya

Early childhood development app helping parents support their child's growth through play-based learning activities.

## Brand
- Parent brand: [[Early Roots]]
- Audience: Parents of children aged 0–5
- Mission: Make developmental milestones accessible and fun

## Tech Stack
- Frontend: React Native, Expo Router, Zustand, RN Reanimated
- Backend: GCP Cloud Run, Firestore
- CI/CD: EAS Build
- Payments: [[RevenueCat]] subscriptions

## Key Metrics
- MRR tracked via [[RevenueCat]]
- App Store ratings and reviews
- User retention (D1, D7, D30)

## Repositories
- `grow-with-freya` — main mobile app
- `colearn` — shared learning content engine
